Aula 14 - Propriedade Display:

é a propriedade CSS mais importante para controlar o layout, especifica se/como um elemento é exibido.

---------------------------------------------------------------------------------------------------------------------------------------

display-block - sempre começa em uma nova linha e ocupa toda a largura disponível.
Exemplos: div, h1 - h6, p, form, header, footer, section.

---------------------------------------------------------------------------------------------------------------------------------------

display-inline - não começa em uma nova linha e ocupa apenas a largura necessária.
Exemplos: span, a, img, link.

---------------------------------------------------------------------------------------------------------------------------------------

display-none - é normalmente usado com JavaScript para ocultar e mostrar elementos sem excluí-los e recriá-los.
Ex:ao passar o mouse sobre o paragrafo.

---------------------------------------------------------------------------------------------------------------------------------------

display-inline-block - ocupam a mesma linha mas com comportamento de bloco.

    Ex:
  • Item 1
  • Item 2
  • Item 3

visibility: hidden - esconde um elemento e ainda ocupará o mesmo espaço de antes.
Ex: passe o mouse sobre o Item 2.
