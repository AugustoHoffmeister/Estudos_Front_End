Font

Aula 12 - Propriedade Font
  • Escolher a fonte certa tem um grande impacto em como os leitores experimentam um site.
  • A fonte certa pode criar uma identidade forte para sua marca.
  • Usar uma fonte fácil de ler é importante.
  • A fonte agrega valor ao seu texto.
  • Também é importante escolher a cor e o tamanho do texto corretos para a fonte.
  • font-family - especifica a fonte de um texto.
  • font web safe - fontes instaladas universalmente em todos os navegadores e dispositivos.
  • font fallbacks - fontes comumente usados, organizados pelas 5 famílias de fontes genéricas.(serif, sans-serif, monospace, cursive, fantasy)
  • font-style - estiliza a fonte.(itálico, oblique)
  • font-size - define o tamanho do texto.
  • font Google - Se você não deseja usar nenhuma das fontes padrão em HTML, pode usar o Google Fonts
  • font pairings - combinação de fontes traz harmonia, sem ser muito parecida ou muito diferente.
  • font shorthand - é possível especificar todas as propriedades de fonte individuais em uma propriedade.

    • * No shorthand os valores font-size e font-family são obrigatórios. Se um dos outros valores estiver ausente, seu valor padrão será usado.
