Aula 03 - Gerenciamento de Versões

git checkout: usado para alternar entre branches, criar novas branches, ou mudar para uma versão anterior do código em um branch existente.

git branch: alista as branches do projeto.

git merge: permite unir o conteúdo de duas ou mais branches em uma única branch. É usado para integrar alterações feitas em uma branch secundária (uma branch de desenvolvimento) na branch principal (master)
