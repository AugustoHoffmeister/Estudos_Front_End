Conceitos

O que são Objetos?

  • Estrutura de dados que contém várias outras estruturas de dados
  • Pode conter métodos
  • Pode conter propriedades
  • Flexível
  • Normalmente usado para guardar registros de dados

O que são Interfaces?

  • Estrutura de dados que são usadas para "tipar" objetos ou tipo de dados em TypeScript
