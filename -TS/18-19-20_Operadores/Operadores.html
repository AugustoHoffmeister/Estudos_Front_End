Operadores de Atribuição

  • Atribuição - x = y │ x = y
  • Adição - x += y │ x = x + y
  • Subtração - x-= y │ x = x - y
  • Multiplicação - x *= y│ x = x * y
  • Divisão - x /= y│ x = x / y
  • Resto - x%= y │ x = x % y
  • Exponencial - x**=y │ x = x ** y

Operadores de Comparação

  • Iguais
  • Não Iguais
  • Estritamente Iguais
  • Estritamente não Iguais
  • Maior que (>)
  • Maior ou Igual (>=)
  • Menor que (<)
  • Menor ou Igual (<=)

Operadores Aritméticos

  • Incremento(++) │ ++i , i++
  • Decremento(--) │ --i , i--
  • Negação(-) │ Se x é 3, então -x retorna -3
  • Adição │ +'3' retorna 3. +true retorna 1,
  • Exponenciação │ 2**3 retorna 8

Importante => Precedência

* A ordem de execução dos operadores no TS é igual a regra matemática
